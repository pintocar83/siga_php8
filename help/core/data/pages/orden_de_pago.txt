==== Registro de Pago ====

Para ingresar al módulo de orden de pago, debemos seleccionar la opción orden de pago desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_11_57_42.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_11_57_51.png?nolink&600|}}

**(1) realizar nuevas órdenes de pago:**

(a) Para ralizar nuevas órdenes de pago, debemos llenar los campos que se presentan: número, fecha, tipo de documento, seleccionar si se realizará a un proveedor o un beneficiario y concepto.

(b) Agregar detalles presupuestario a la orden de pago:

Al hacer click en el botón agregar {{:captura_de_pantalla_de_2015-05-25_10_18_18.png?nolink&80|}} se abrirá una ventana donde se podrá llenar los detalles presupuestarios. 
  
{{:captura_de_pantalla_de_2015-05-25_12_16_50.png?nolink&600|}}

( c ) Opcionalmente si agregó un detalle presupuestario y no lo desea puede quitarlo de la siguiente manera.

- Seleccionar el detalle presupuestario a eliminar de la orden de pago.
- Hacer click en el botón quitar {{:captura_de_pantalla_de_2015-05-25_10_25_28.png?nolink&80|}} posteriormente se habrá eliminado de ésta.

(b) Se procede a pulsar el botón guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_11_59_57.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-25_12_00_02.png?nolink&600|}}

**(2) Busqueda de orden de pago:**

(a) Seleccionar la pestaña listado en la pantalla principal.

(b) Escribir el nombre del proveedor, beneficiario o Número en el campo búscar.

{{:captura_de_pantalla_de_2015-05-25_12_02_24.png?nolink&600|}}

( c ) Seleccionamos los servicios del listado, automáticamente el formulario de datos se carga con la información de las órdenes de pago registradas.

{{:captura_de_pantalla_de_2015-05-25_12_03_08.png?nolink&600|}}

**(3) Editar Orden de Pago:**

( a ) Debemos realizar el paso 2 (Búsqueda de órden de pago a editar).

( b ) Presionamos el botón Modificar en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_12_21_02.png?nolink&600|}}
( c ) Modificamos los campos necesarios.

( d ) Presionamos el botón Guardar.


**(4) Eliminar las órdenes de pago:**

( a ) Debemos realizar el paso 2 (Búsqueda de las órdenes pagos a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_12_23_32.png?nolink&600|}}

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

NOTA: Esta opción se encuentra deshabilitada.
