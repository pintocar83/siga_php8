__**Registro de Pago**__

Para ingresar al módulo de orden de pago, debemos seleccionar la opción orden de pago desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_11_57_42.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_11_57_51.png?nolink&600|}}

**(1) realizar nuevas ordenes de pago:**

(a) Para ralizar nuevas ordenes de pago, debemos llenar los campos que se presentan: número, fecha, tipo de documento, seleccionar si se realizará a un proveedor o un beneficiario y concepto.

(b) Se procede a pulsar el boton guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_11_59_57.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-25_12_00_02.png?nolink&600|}}

**(2) Busqueda de orden de pago:**

(a) Seleccionar la pestaña listado en la pantalla principal.

(b) Escribir el nombre del servicio o Código en el campo buscar.

{{:captura_de_pantalla_de_2015-05-25_12_02_24.png?nolink&600|}}

( c ) Seleccionamos los servicios del listado, automáticamente el formulario de datos se carga con la información de las ordenes de pago registradas.

{{:captura_de_pantalla_de_2015-05-25_12_03_08.png?nolink&600|}}

**(3) Editar Orden de Pago:**

( a ) Debemos realizar el paso 2 (Busqueda de orden de pago a editar).

( b ) Presionamos el botón Modificar en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_12_09_52.png?nolink&600|}}

( c ) Modificamos los campos necesarios.

( d ) Presionamos el botón Guardar.


**(4) Eliminar las ordenes de pago:**

( a ) Debemos realizar el paso 2 (Busqueda de las ordenes pagos a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

NOTA: Esta opción se encuentra deshabilitada.
