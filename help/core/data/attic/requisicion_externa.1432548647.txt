__**Requisición Externa**__

Para ingresar al módulo de Requisición Externa, debemos seleccionar la opción Requisición Externa desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_10_00_19.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_10_01_32.png?nolink&600|}}

**(1) Registrar Nuevas Requisiciones:**

(a) Para registrar nuevas Requisiciones, debemos llenar los campos que se presentan: Requisición, Fecha, Correlativo, estructura presupuestaria, concepto.

(b) Agragar item a la requisición:

Al hacer click en el botom agregar se abrirá una ventana donde se listaran los item según sea el caso. 
  
{{:captura_de_pantalla_de_2015-05-25_10_03_30.png?nolink&600|}}

( c ) Se procede a pulsar el boton guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_10_06_14.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-25_10_08_02.png?nolink&600|}}

**(2) Busqueda de Requisición:**

(a) Seleccionar la pestaña listado en la pantalla principal.

(b) Escribir el nombre del servicio o Código en el campo buscar.

{{:captura_de_pantalla_de_2015-05-25_10_09_30.png?nolink&600|}}

( c ) Seleccionamos los servicios del listado, automáticamente el formalario de datos se carga con la información de los bienes registrado.

{{:captura_de_pantalla_de_2015-05-25_10_09_48.png?nolink&600|}}

**(3) Eliminar los Servicios:**

( a ) Debemos realizar el paso 2 (Busqueda de los Servicios a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.
