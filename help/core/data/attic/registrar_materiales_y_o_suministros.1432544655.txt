__**Registro de Materiales y/o Suministros**__

Para ingresar al módulo de registros de materiales y/o Suministros, debemos seleccionar la opción Materiales y/o Suministros desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_08_24_19.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_08_27_15.png?nolink&600|}}

**(1) Registrar Nuevo Material y/o Suministro:**

(a) Para registrar un nuevo Material y/o Suministros, debemos llenar los campos que se presentan: Código, Denominación, Partida presupuestaria y seleccionar si aplica I.V.A o no.

(b) Se procede a pulsar el boton guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_08_34_09.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-25_08_40_52.png?nolink&600|}}

**(2) Busqueda de Materiales y/o Suministros:**

(a) Seleccionar la pestaña listado en la pantalla principal.

{{:captura_de_pantalla_de_2015-05-25_08_44_42.png?nolink&600|}}

(b) Escribir el nombre del materiales y/o Suministros o Código en el campo buscar.

{{:captura_de_pantalla_de_2015-05-25_08_50_06.png?nolink&600|}}

( c ) Seleccionamos el Material y/o Suministro del listado, automáticamente el formalario de datos de carga con la información del material y/o suministro registrado.

{{:captura_de_pantalla_de_2015-05-25_08_52_19.png?nolink&600|}}

**(3) Eliminar Material y/o Suministro:**

( a ) Debemos realizar el paso 2 (Busqueda del material y/o suministro a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

{{:captura_de_pantalla_de_2015-05-25_08_57_47.png?nolink&600|}}
