__**Orden de Servicio**__

Para ingresar al módulo de orden de servicio, debemos seleccionar la opción Orden de Servicio desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_11_28_17.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_11_28_24.png?nolink&600|}}

**(1) Elaborar nuevas órdenes de servicios:**

(a) Para elaborar nuevas órdenes de servicios, debemos llenar los campos que se presentan: número, fecha, proveedor, concepto, porciento de descuento.

{{:captura_de_pantalla_de_2015-05-25_11_36_33.png?nolink&600|}}

**(2) Agregar requisiciones:**

Al hacer click en el botón requisiciones {{:requisicion.png?nolink&120|}} se abrirá una ventana donde se listaran las requisiciones pendientes. 
  
{{:captura_de_pantalla_de_2015-05-25_11_38_03.png?nolink&600|}}

(a) opcionalmente si la compra posee cargos se le da click al botón cargos {{:captura_de_pantalla_de_2015-05-25_11_02_28.png?nolink&70|}} la cual despliega una ventana donde se eligirá el cargo necesario.

{{:captura_de_pantalla_de_2015-05-25_11_38_59.png?nolink&600|}}

(b) Se procede a pulsar el botón guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_11_40_41.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-25_11_42_26.png?nolink&600|}}

**(3) Búsqueda de orden de servicio:**

(a) Seleccionar la pestaña listado en la pantalla principal.

(b) Escribir el nombre del proveedor de la orden de servicio o el número en el campo búscar.

{{:captura_de_pantalla_de_2015-05-25_11_44_15.png?nolink&600|}}

( c ) Seleccionamos los servicios del listado, automáticamente el formulario de datos se carga con la información de las órdenes de servicios registradas.

{{:captura_de_pantalla_de_2015-05-25_11_44_52.png?nolink&600|}}

**(4) Editar Orden de Servicios:**

( a ) Debemos realizar el paso 3 (Búsqueda de orden de servicio a editar).

( b ) Presionamos el botón Modificar en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_11_46_36.png?nolink&600|}}

( c ) Modificamos los campos necesarios.

( d ) Presionamos el botón Guardar.



**(5) Eliminar orden de servicio:**

( a ) Debemos realizar el paso 3 (Búsqueda de orden de sevicio a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

NOTA: Esta opción se encuentra deshabilitada.
