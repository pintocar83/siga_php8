__**REGISTAR BENEFICIARIO**__


  **      
  - Ítem de lista ordenada 

  **




Para ingresar al módulo para la creación y edición de los beneficiarios, debemos seleccionar la opción "Beneficiarios" desde el menú principal del sistema administrativo.


{{:captura_de_pantalla_de_2015-05-22_11_27_55.png?nolink&600|}}


A continuación se despliega la ventana referente al módulo.


{{:captura_de_pantalla_de_2015-05-22_11_31_03.png?nolink&600|}}




- ** Registrar Nuevo Beneficiario:**

    - Para registrar un nuevo beneficiario, debemos llenar los campos que se presentan: Cédula,Primer nombre, Segundo nombre,Primer apellido,Segundo apellido,Teléfono y Correo.


(b) Se procede a pulsar el botón guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-22_11_35_08.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito.', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-22_11_38_38.png?nolink&600|}}

**(2) Búsqueda de Beneficiarios Registrados:**

(a) Seleccionar la pestaña "Lista" en la pantalla principal.

{{:captura_de_pantalla_de_2015-05-22_11_41_49.png?nolink&600|}}

(b) Escribir el nombre del beneficiario o RIF en el campo buscar.

{{:captura_de_pantalla_de_2015-05-22_11_43_52.png?nolink&600|}}

( c ) Seleccionamos el beneficiario del listado, automáticamente el formulario de datos se carga con la información de los beneficiarios registrados.

{{:captura_de_pantalla_de_2015-05-22_11_49_11.png?nolink&600|}}

**(3) Editar Beneficiario:**

( a ) Debemos realizar el paso 2 (Busqueda del beneficiario a editar).

( b ) Presionamos el botón Modificar en la barra superior.

( c ) Modificamos los campos necesarios.

( d ) Presionamos el botón Guardar.

{{:captura_de_pantalla_de_2015-05-22_11_50_58.png?nolink&600|}}

**( 4 ) Eliminar Beneficiario:**

( a ) Debemos realizar el paso 2 (Busqueda del beneficiario a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro que desea eliminarlo?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

{{:captura_de_pantalla_de_2015-05-22_11_51_32.png?600|}}
