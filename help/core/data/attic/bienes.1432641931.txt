__**Registro de Bienes**__

Para ingresar al módulo de registros de Bienes, debemos seleccionar la opción "Bienes" desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_09_11_48.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_09_11_52.png?nolink&600|}}

**(1) Registrar Nuevos Bienes:**

(a) Para registrar nuevos bienes, debemos llenar los campos que se presentan: Código, Denominación, Partida presupuestaria y seleccionar si aplica I.V.A o no.

(b) Se procede a pulsar el botón guardar, ubicado en la barra superior.

{{:captura_de_pantalla_de_2015-05-25_09_11_59.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 

{{:captura_de_pantalla_de_2015-05-25_09_12_36.png?nolink&600|}}

**(2) Búsqueda de Bienes:**

(a) Seleccionar la pestaña "Lista" en la pantalla principal.

(b) Escribir el nombre de los Bienes o Código en el campo buscar.

{{:captura_de_pantalla_de_2015-05-25_09_12_15.png?nolink&600|}}

( c ) Seleccionamos los bienes del listado, automáticamente el formulario de datos de carga con la información de los bienes registrados.

{{:captura_de_pantalla_de_2015-05-25_09_12_19.png?nolink&600|}}

**(3) Eliminar los Bienes:**

( a ) Debemos realizar el paso 2 (Búsqueda de los bienes a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

{{:captura_de_pantalla_de_2015-05-25_09_13_34.png?nolink&600|}}
