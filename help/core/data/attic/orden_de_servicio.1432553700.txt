__**Orden de Servicio**__

Para ingresar al módulo de orden de servicio, debemos seleccionar la opción Orden de Servicio desde el menú principal del sistema administrativo.

{{:captura_de_pantalla_de_2015-05-25_11_28_17.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_de_pantalla_de_2015-05-25_11_28_24.png?nolink&600|}}

**(1) Elaborar nuevas ordenes de servicios:**

(a) Para elaborar nuevas ordenes de servicios, debemos llenar los campos que se presentan: número, fecha, proveedor, comcepto, porciento de descuento.



**(2) Agregar requisiciones:**

Al hacer click en el botom requisiciones {{:requisicion.png?nolink&120|}} se abrirá una ventana donde se listaran las requisiciones pendientes. 
  


(a) opcionalmente si la compra posee cargos se le da click al botom cargos {{:captura_de_pantalla_de_2015-05-25_11_02_28.png?nolink&70|}} la cual despliega una ventana donde se eligirá el cargo necesario.



(b) Se procede a pulsar el boton guardar, ubicado en la barra superior.



A continuación se muestra el mensaje 'Datos Guardados con exito', si no existen errores en la carga de información en el formulario. 


**(3) Búsqueda de orden de servicio:**

(a) Seleccionar la pestaña listado en la pantalla principal.

(b) Escribir el nombre del proveedor de la orden de servicio o el numero en el campo buscar.



( c ) Seleccionamos los servicios del listado, automáticamente el formulario de datos se carga con la información de las ordenes de servicios registradas.



**(4) Editar Orden de Servicios:**

( a ) Debemos realizar el paso 3 (Busqueda de orden de servicio a editar).

( b ) Presionamos el botón Modificar en la barra superior.



( c ) Modificamos los campos necesarios.

( d ) Presionamos el botón Guardar.



**(5) Eliminar orden de servicio:**

( a ) Debemos realizar el paso 3 (Busqueda de orden de sevicio a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro de eliminar el registro?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.

NOTA: Esta opción se encuentra deshabilitada.
