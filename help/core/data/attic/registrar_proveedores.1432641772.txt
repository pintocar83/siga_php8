__**Registro de Proveedores**__

Para ingresar al módulo para la creación y edición de los proveedores, debemos seleccionar la opción "Proveedores" desde el menú principal del sistema administrativo.

{{:captura_el_2015-05-22_a_las_10.08.10.png?nolink&600|}}

A continuación se despliega la ventana referente al módulo.

{{:captura_el_2015-05-22_a_las_10.20.14.png?nolink&600|}}

**(1) Registra Nuevo Proveedor:**

(a) Para registrar un nuevo proveedor, debemos llenar los campos que se presentan: RIF, Denominación, Telefono, Correo y Dirección.

(b) Se procede a pulsar el boton guardar, ubicado en la barra superior.

{{:captura_el_2015-05-22_a_las_10.26.50.png?nolink&600|}}

A continuación se muestra el mensaje 'Datos Guardados con exito.', si no existen errores en la carga de información en el formulario. 

{{:captura_el_2015-05-22_a_las_10.33.56.png?nolink&600|}}

**(2) Búsqueda de Proveedores Registrados:**

(a) Seleccionar la pestaña "Lista" en la pantalla principal.

{{:captura_el_2015-05-22_a_las_10.44.00.png?nolink&600|}}

(b) Escribir el nombre del proveedor o RIF en el campo buscar.

( c ) Seleccionamos el proveedor del listado, automáticamente el formulario de datos se carga con la información del proveedor registrado.

{{:captura_el_2015-05-22_a_las_10.49.06.png?nolink&600|}}

{{:captura_el_2015-05-22_a_las_10.49.41.png?nolink&600|}}

**(3) Editar Proveedor:**

( a ) Debemos realizar el paso 2 (Busqueda del proveedor a editar).

( b ) Presionamos el botón Modificar en la barra superior.

( c ) Modificamos los campos necesarios.

( d ) Presionamos el botón Guardar.

{{:captura_de_pantalla_de_2015-05-22_11_15_06.png?nolink&600|}}

**(4) Eliminar Proveedor:**

( a ) Debemos realizar el paso 2 (Busqueda del proveedor a eliminar).

( b ) Presionamos el botón Eliminar en la barra superior.

( c ) El sistema nos consulta: ¿Está seguro que desea eliminarlo?
si aceptamos, el registro es eliminado. Por el contrario, cancelamos la operación.


{{:captura_de_pantalla_de_2015-05-22_11_18_00.png?nolink&600|}}
